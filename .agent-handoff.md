## Quick Dev Workflow (for new machine)
1. `git clone https://github.com/AspicyKabob/-agentaudit-api.git`
2. `cd -agentaudit-api`
3. `npm install`
4. `cp .env.example .env`
5. Fill in `.env` with dev values (see `.env.example` for structure)
6. `npx prisma migrate dev`
7. `npm run dev`
